Визуализация множества Мандельброта с возможностью масштабирония любого его участка.
Клик левой кнопкой мыши - увеличение участка.
Клик правой кнопкой мыши - уменьшение участка.
Клие средней кннопкой мыши - возврат к исходному состоянию, которое было при запуске.
